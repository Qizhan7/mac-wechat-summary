============================================
    微信群聊 AI 总结 - 使用说明
============================================


【安装（只需一次）】

第一步：解压后双击「启动.command」

  找到解压出来的文件夹，双击「启动.command」文件。
  不要只单独拷贝某一个文件，整个文件夹要保留在一起。


---------------------------------------------
⚠ 遇到安全提示怎么办？
---------------------------------------------

macOS 会拦截从网上下载的文件，这是正常的。按下面的方法解除：

  方法一（推荐）：右键打开

    1. 在「启动.command」文件上 右键（或两根手指点触控板）
    2. 选择「打开」
    3. 弹窗里再点「打开」

    * 只有第一次需要这样做，以后双击就能直接打开。

  方法二：从系统设置解除

    如果方法一没用：

    1. 打开 系统设置 → 隐私与安全性
    2. 往下滚，找到「启动.command 已被阻止」的提示
    3. 点右边的「仍要打开」
    4. 输入电脑密码确认


---------------------------------------------
第二步：等待自动配置
---------------------------------------------

首次运行会自动完成以下步骤（大约 2-5 分钟）：

  1. 在当前文件夹创建独立的 Python 环境（.venv）
  2. 安装 Python 依赖
  3. 检查/修复微信授权（需要输入电脑密码）
  4. 完成！

  * .venv 是项目自己的运行环境，不会污染你电脑里别的 Python。
  * 微信授权是为了读取聊天记录数据库，不影响微信正常使用。


---------------------------------------------
第三步：设置 API Key
---------------------------------------------

  1. 菜单栏出现 💬 图标后（在屏幕上方），点击它
  2. 选择「设置」
  3. 填入你的 AI 服务 API Key

支持的 AI 服务：（强烈推荐千问，免费额度多。本地模型如果配置不够好很慢，不推荐）

  通义千问（默认）  → dashscope.console.aliyun.com
  DeepSeek          → platform.deepseek.com
  Claude            → console.anthropic.com
  OpenAI            → platform.openai.com
  Ollama 本地       → 无需 Key，需先安装 Ollama



============================================
    日常使用
============================================

  1. 确保微信已启动并登录
  2. 双击「启动.command」
  3. 菜单栏出现 💬 图标
  4. 点击图标 → 选择群聊 → 查看/生成总结

配置文件位置：
  ~/.wechat-summary/config.json

总结和缓存位置：
  ~/.wechat-summary/



============================================
    常见问题
============================================

Q：提示「未找到 Python3」或「Python 版本太低」？
A：本工具需要 Python 3.10 以上版本。
   启动脚本会自动检测版本，如果版本太低且已安装 Homebrew，
   会尝试自动安装 Python 3.12。
   如果自动安装不成功，请手动安装：
   · 方法一（推荐）：去 python.org/downloads 下载最新版本
   · 方法二：运行 brew install python@3.12

Q：提示「重签名失败」/ Operation not permitted？
A：这是因为 macOS 没有授权终端修改应用。
   1. 先完全退出微信
   2. 打开「系统设置」→「隐私与安全性」
   3. 往下滚，找到「App 管理」
   4. 打开「终端」的开关（需要输入密码）
   5. 重新双击「启动.command」
   如果找不到「App 管理」，就找「完全磁盘访问权限」，打开「终端」的开关。

Q：终端提示输入密码，但打字看不见，是不是卡住了？
A：不是。macOS 的 sudo 输入密码时本来就不会显示字符，也不会显示星号。
   直接输入电脑登录密码，然后按回车即可。

Q：微信更新后读不到记录了？
A：微信每次更新后，脚本都会自动检查授权状态。
   直接重新双击「启动.command」即可，它会提示并自动处理。

Q：关闭终端窗口会怎样？
A：关闭终端窗口会退出程序。想退出时直接关窗口或按 Ctrl+C。

Q：数据存在哪里？
A：配置、密钥缓存、总结记录都在 ~/.wechat-summary/ 目录。
   Python 依赖环境在当前项目文件夹的 .venv/ 目录。

Q：通过 MCP 发送微信消息失败？
A：发送消息需要辅助功能权限。
   1. 打开「系统设置」→「隐私与安全性」→「辅助功能」
   2. 找到你运行 MCP Server 的应用（终端、Claude.app 等）
   3. 打开它的开关（需要输入密码）
   4. 重新启动该应用
   发送时微信必须已登录且在运行中，AI 会自动操作微信界面完成发送，
   发送后会自动验证消息是否成功写入数据库。

Q：我要把这个工具转发给别人，怎么打包？
A：直接压缩整个项目文件夹即可。
   如果你本机已经跑过一次，压缩前建议先删掉项目里的 .venv/ 目录，再打包发出去。
